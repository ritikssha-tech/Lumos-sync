# Lumos-Sync ☀️
**Smart-Integrated Solar Alignment System**

Lumos-Sync is a mobile-responsive web application designed to bridge the gap between physical solar reflectors and digital intelligence. Using mobile device sensors, it helps users find the optimal angle for solar reflectors to maximize energy efficiency.

## 🚀 Features
- **Real-time Alignment:** Uses device orientation sensors for precise positioning.
- **Eco-Friendly:** Designed to promote sustainable energy usage.
- **Minimalist UI:** Simple, clean interface for ease of use in outdoor conditions.

## 🛠️ Tech Stack
- HTML5, CSS3, JavaScript
- Vercel (Deployment)
- Cursor AI & VS Code (Development)
